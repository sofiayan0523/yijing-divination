# Tech Innovation

A bold and modern theme with high-contrast colors for cutting-edge presentations.

## Color Palette
- **Electric Blue**: `#0066ff` - Vibrant primary accent
- **Neon Cyan**: `#00ffff` - Bright highlight color
- **Dark Gray**: `#1e1e1e` - Deep backgrounds
- **White**: `#ffffff` - Clean text and contrast

## Typography
- **Headers**: DejaVu Sans Bold
- **Body Text**: DejaVu Sans

## Best Used For
Tech startups, software launches, innovation showcases, AI/ML presentations.
