# Arctic Frost

A cool and crisp winter-inspired theme that conveys clarity and professionalism.

## Color Palette
- **Ice Blue**: `#d4e4f7` - Light backgrounds and highlights
- **Steel Blue**: `#4a6fa5` - Primary accent color
- **Silver**: `#c0c0c0` - Metallic accent elements
- **Crisp White**: `#fafafa` - Clean backgrounds and text

## Typography
- **Headers**: DejaVu Sans Bold
- **Body Text**: DejaVu Sans

## Best Used For
Healthcare presentations, technology solutions, clean tech, pharmaceutical content.
